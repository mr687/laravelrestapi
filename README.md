# Assignment 7 - Kampus Merdeka
Simple Online Shop View.

## Clone and Run
```sh
$ composer install
$ cp .env.example .env
$ php artisan key:generate
$ php artisan serve
```

Endpoint: `http://127.0.0.1:8000`
